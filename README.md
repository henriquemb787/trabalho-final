Como surgiu
Tentamos achar um projeto pra trabalho final que não fosse muito difícil pra fazer e que englobe tudo
que nos estudamos nas aulas 
Para que serve
O sistema permite ao usuário criar uma conta bancária acessar sua conta e gerenciar seu saldo por
meio de operações básicas
Funcionalidade
Criar uma conta bancária.
Fazer login em uma conta cadastrada.
Consultar o saldo da conta.
Realizar depósitos.
Estruturas que serão utilizadas
cadastro e exibição das informações.
verificar se o login está correto e validar as operações.
manter o menu funcionando até o usuário escolher sair.
armazenar as contas cadastradas.
guardar os dados de cada conta, como nome, senha e saldo.
organizar cada funcionalidade do sistema em uma função específica.
